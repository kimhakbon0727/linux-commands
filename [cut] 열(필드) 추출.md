# ✂️ cut

텍스트의 각 줄에서 **특정 열(필드) 또는 문자 위치를 잘라내는** 명령어.

---

## 기본 문법

```bash
cut [옵션] [파일]
```

---

## 구분자 기준으로 열 추출 (-d, -f)

```bash
# ':' 구분자로 나눈 1번째 열 추출
cut -d':' -f1 /etc/passwd

# ',' 구분자로 나눈 2번째 열 추출
cut -d',' -f2 data.csv

# 탭 구분자로 1, 3번째 열 추출
cut -d$'\t' -f1,3 file.tsv

# 2번째부터 4번째 열까지 추출
cut -d',' -f2-4 data.csv

# 3번째 열 이후 전부 추출
cut -d':' -f3- /etc/passwd

# 1번째부터 3번째 열까지 추출
cut -d':' -f-3 /etc/passwd
```

---

## 문자 위치 기준 추출 (-c)

```bash
# 1번째 문자만 추출
cut -c1 file.txt

# 1~5번째 문자 추출
cut -c1-5 file.txt

# 3번째 이후 문자 추출
cut -c3- file.txt

# 1, 3, 5번째 문자 추출
cut -c1,3,5 file.txt
```

---

## 바이트 기준 추출 (-b)

```bash
# 1~10번째 바이트 추출
cut -b1-10 file.txt
```

> 멀티바이트 문자(한글 등)가 있으면 `-c` 대신 `-b` 사용 시 깨질 수 있음

---

## 자주 쓰는 조합

```bash
# /etc/passwd 에서 사용자명만 추출
cut -d':' -f1 /etc/passwd

# CSV에서 특정 열 추출
cut -d',' -f1,3 data.csv

# IP 주소에서 서브넷 부분만 추출
echo "192.168.1.100" | cut -d'.' -f1-3

# 파일 확장자 추출
echo "document.pdf" | cut -d'.' -f2

# 로그에서 날짜 부분만 추출
cut -d' ' -f1 access.log | sort | uniq -c
```

---

## 주요 옵션 정리

| 옵션 | 설명 |
|------|------|
| `-d '구분자'` | 필드 구분자 지정 (기본값: 탭) |
| `-f N` | N번째 필드 추출 |
| `-f N,M` | N, M번째 필드 추출 |
| `-f N-M` | N~M번째 필드 추출 |
| `-f N-` | N번째 이후 전부 |
| `-f -N` | N번째까지 전부 |
| `-c N` | N번째 문자 추출 |
| `-b N` | N번째 바이트 추출 |

---

## cut vs awk 비교

| 상황 | 추천 |
|------|------|
| 단순 열 추출 | `cut` (빠르고 간단) |
| 구분자가 여러 개의 공백 | `awk` (공백 정규화 가능) |
| 조건부 필터링 필요 | `awk` |
| CSV 단순 열 추출 | `cut` |

---

## 팁

- 구분자 기본값은 **탭** 이므로 공백 구분이면 `-d' '` 명시 필요
- 공백이 여러 개인 경우 `cut` 으로 처리하기 어려우므로 `awk` 사용 권장
- `grep` → `cut` 조합으로 원하는 값만 빠르게 추출 가능
