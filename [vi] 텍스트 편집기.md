# ✏️ 텍스트 편집기 - vi 명령어

> 리눅스 환경 어디서나 사용 가능한 **기본 텍스트 편집기**  
> 익숙해지면 마우스 없이도 빠르게 파일을 편집할 수 있는 필수 도구

---

## 📌 기본 문법

```bash
vi [옵션] [파일명]
```

---

## ⚙️ 주요 옵션 (실행 전)

| 옵션 | 설명 |
|------|------|
| `+숫자` | 지정한 줄 번호로 바로 이동해서 열기 |
| `+/패턴` | 해당 패턴이 있는 줄로 바로 이동해서 열기 |
| `-R` | 읽기 전용(Read-Only)으로 열기 |
| `-r 파일` | 비정상 종료된 파일 복구 |
| `-n` | swap 파일 없이 열기 |

---

## 💡 기본 사용 예시

```bash
# 새 파일 생성 또는 기존 파일 열기
vi filename.txt

# 특정 줄 번호로 바로 이동
vi +10 filename.txt

# 특정 단어가 있는 줄로 바로 이동
vi +/error filename.txt

# 읽기 전용으로 열기
vi -R filename.txt

# 여러 파일 동시에 열기
vi file1.txt file2.txt
```

---

## 🖥️ vi의 3가지 모드

vi는 **모드 기반** 편집기입니다. 현재 모드에 따라 키 동작이 완전히 달라집니다.

```
┌─────────────────────────────────────────────┐
│                                             │
│   일반 모드 ──────i, a, o──────▶ 입력 모드  │
│   (Normal)  ◀──────── ESC ────────          │
│       │                                     │
│       └──────── : ─────────▶ 명령 모드      │
│                              (Command)      │
└─────────────────────────────────────────────┘
```

| 모드 | 진입 방법 | 용도 |
|------|-----------|------|
| **일반 모드** (Normal) | 시작 시 기본 / `ESC` | 커서 이동, 복사, 삭제, 검색 |
| **입력 모드** (Insert) | `i`, `a`, `o` 등 | 텍스트 직접 입력 |
| **명령 모드** (Command) | `:` | 저장, 종료, 치환, 설정 |

> 💡 **모르면 일단 `ESC` 여러 번** — 항상 일반 모드로 돌아옵니다

---

## ⌨️ 일반 모드 단축키

### 🔀 커서 이동

| 키 | 동작 |
|----|------|
| `h` | ← 왼쪽 |
| `j` | ↓ 아래 |
| `k` | ↑ 위 |
| `l` | → 오른쪽 |
| `w` | 다음 단어 첫 글자로 이동 |
| `b` | 이전 단어 첫 글자로 이동 |
| `e` | 현재 단어 마지막 글자로 이동 |
| `0` | 줄의 맨 앞으로 이동 |
| `$` | 줄의 맨 끝으로 이동 |
| `gg` | 파일의 맨 첫 줄로 이동 |
| `G` | 파일의 맨 마지막 줄로 이동 |
| `숫자G` | 해당 줄 번호로 이동 (예: `10G` → 10번 줄) |
| `Ctrl + f` | 한 페이지 아래로 이동 |
| `Ctrl + b` | 한 페이지 위로 이동 |
| `Ctrl + d` | 반 페이지 아래로 이동 |
| `Ctrl + u` | 반 페이지 위로 이동 |

---

### ✍️ 입력 모드 진입

| 키 | 동작 |
|----|------|
| `i` | 커서 앞에서 입력 시작 |
| `I` | 줄의 맨 앞에서 입력 시작 |
| `a` | 커서 뒤에서 입력 시작 |
| `A` | 줄의 맨 끝에서 입력 시작 |
| `o` | 현재 줄 아래에 새 줄 추가 후 입력 |
| `O` | 현재 줄 위에 새 줄 추가 후 입력 |
| `s` | 현재 글자 삭제 후 입력 모드 |
| `S` | 현재 줄 전체 삭제 후 입력 모드 |
| `ESC` | 입력 모드 종료 → 일반 모드로 복귀 |

---

### 🗑️ 삭제

| 키 | 동작 |
|----|------|
| `x` | 커서 위치의 글자 1개 삭제 |
| `X` | 커서 앞의 글자 1개 삭제 (Backspace) |
| `dd` | 현재 줄 전체 삭제 |
| `숫자dd` | 현재 줄부터 n줄 삭제 (예: `3dd`) |
| `dw` | 현재 단어 삭제 |
| `d$` | 커서부터 줄 끝까지 삭제 |
| `d0` | 커서부터 줄 앞까지 삭제 |
| `dG` | 커서부터 파일 끝까지 삭제 |
| `dgg` | 커서부터 파일 처음까지 삭제 |

---

### 📋 복사 & 붙여넣기

| 키 | 동작 |
|----|------|
| `yy` | 현재 줄 복사 (yank) |
| `숫자yy` | 현재 줄부터 n줄 복사 (예: `3yy`) |
| `yw` | 현재 단어 복사 |
| `y$` | 커서부터 줄 끝까지 복사 |
| `p` | 커서 아래(뒤)에 붙여넣기 |
| `P` | 커서 위(앞)에 붙여넣기 |

---

### ↩️ 실행 취소 & 재실행

| 키 | 동작 |
|----|------|
| `u` | 실행 취소 (Undo) |
| `U` | 현재 줄 전체 변경 취소 |
| `Ctrl + r` | 재실행 (Redo) |

---

### 🔍 검색

| 키 | 동작 |
|----|------|
| `/패턴` | 아래 방향으로 검색 |
| `?패턴` | 위 방향으로 검색 |
| `n` | 다음 검색 결과로 이동 |
| `N` | 이전 검색 결과로 이동 |
| `*` | 커서 위의 단어를 아래 방향으로 검색 |
| `#` | 커서 위의 단어를 위 방향으로 검색 |

## 📝 명령 모드 (`:` 입력 후)

### 💾 저장 & 종료

| 명령 | 동작 |
|------|------|
| `:w` | 저장 |
| `:w 파일명` | 다른 이름으로 저장 |
| `:q` | 종료 (변경 없을 때) |
| `:q!` | 저장하지 않고 강제 종료 |
| `:wq` | 저장 후 종료 |
| `:wq!` | 강제 저장 후 종료 |
| `ZZ` | 저장 후 종료 (`:wq` 와 동일) |
| `ZQ` | 저장하지 않고 종료 (`:q!` 와 동일) |

---

### 🔄 치환 (Find & Replace)

| 명령 | 동작 |
|------|------|
| `:s/찾기/바꾸기` | 현재 줄 첫 번째만 치환 |
| `:s/찾기/바꾸기/g` | 현재 줄 전체 치환 |
| `:%s/찾기/바꾸기/g` | 파일 전체 치환 |
| `:%s/찾기/바꾸기/gc` | 파일 전체 치환 (하나씩 확인) |
| `:10,20s/찾기/바꾸기/g` | 10~20번 줄 범위에서 치환 |

```bash
# 예시: 파일 전체에서 'hello'를 'world'로 치환
:%s/hello/world/g

# 대소문자 구분 없이 치환
:%s/hello/world/gi
```

---

### 📄 기타 명령

| 명령 | 동작 |
|------|------|
| `:set nu` | 줄 번호 표시 |
| `:set nonu` | 줄 번호 숨기기 |
| `:set hlsearch` | 검색어 하이라이트 |
| `:set nohlsearch` | 검색어 하이라이트 끄기 |
| `:숫자` | 해당 줄로 이동 (예: `:10`) |
| `:e 파일명` | 다른 파일 열기 |
| `:r 파일명` | 현재 커서 아래에 파일 내용 삽입 |
| `:%d` | 파일 전체 내용 삭제 |

## 🛠️ 실전 활용 예시

```bash
# 설정 파일 편집
vi /etc/nginx/nginx.conf

# 특정 줄로 바로 이동해서 열기
vi +50 /var/log/syslog

# 에러 로그 확인 (읽기 전용)
vi -R /var/log/auth.log

# 비정상 종료 후 복구
vi -r filename.txt

# 줄 번호 표시 후 편집 (자주 씀)
vi filename.txt
# 열고 나서 :set nu 입력
```

## 🔁 vi vs vim vs nano 비교

| 항목 | vi | vim | nano |
|------|-----|-----|------|
| 기본 설치 | ✅ 거의 모든 리눅스 | ✅ 대부분 포함 | ✅ 대부분 포함 |
| 문법 강조 | ❌ | ✅ | 🔺 일부 |
| 실행 취소 | 1단계 | ✅ 다단계 | ✅ |
| 학습 난이도 | 높음 | 높음 | 낮음 |
| 플러그인 | ❌ | ✅ | ❌ |
| 추천 상황 | 최소 환경 서버 | 일반 서버 편집 | 빠른 간단 편집 |

```bash
# vim 설치
sudo apt install vim     # Ubuntu/Debian
sudo yum install vim     # CentOS/RHEL

# nano 설치
sudo apt install nano
```

## ⚡ 자주 쓰는 조합 모음

```bash
vi +/error log.txt      # error 단어 있는 줄로 바로 이동
vi -R /etc/passwd       # 읽기 전용으로 시스템 파일 확인
:%s/old/new/gc          # 전체 치환 (하나씩 확인하며)
:set nu                 # 줄 번호 켜기 (편집 시 필수)
gg=G                    # 전체 코드 자동 들여쓰기 (vim)
:10,20d                 # 10~20번 줄 삭제
```

## 🆘 vi 탈출법 (처음 쓸 때 막히는 상황)

> 처음 vi를 쓰면 종료를 못해서 당황하는 경우가 많습니다!

```
1. ESC 키를 누른다 (일반 모드로 복귀)
2. 아래 중 하나 입력:

   :wq   → 저장하고 종료
   :q!   → 저장 안 하고 강제 종료  ← 가장 많이 씀
   ZZ    → 저장하고 종료 (단축키)
```

> 💬 **Tip**: `~/.vimrc` 파일에 설정을 저장하면 vim 실행 시 자동 적용됩니다!
> ```bash
> # ~/.vimrc 기본 추천 설정
> set number          " 줄 번호 표시
> set hlsearch        " 검색어 하이라이트
> set tabstop=4       " 탭 크기 4칸
> set autoindent      " 자동 들여쓰기
> set syntax on       " 문법 강조
> ```
