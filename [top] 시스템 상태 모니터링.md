# 🖥️ 실시간 모니터링 - top 명령어

> 시스템의 **CPU, 메모리, 프로세스 상태를 실시간으로 모니터링**하는 명령어  
> 리눅스 관리자라면 가장 먼저 익혀야 할 필수 도구

---

## 📌 기본 문법

```bash
top [옵션]
```

---

## ⚙️ 주요 옵션 (실행 전)

| 옵션 | 설명 |
|------|------|
| `-d 초` | 갱신 주기 설정 (기본값: 3초) |
| `-n 횟수` | 지정 횟수만큼 실행 후 종료 |
| `-p PID` | 특정 프로세스만 감시 |
| `-u 유저` | 특정 유저의 프로세스만 표시 |
| `-b` | 배치 모드 (파일 저장/파이프 연동용) |
| `-i` | 유휴(idle) 프로세스 숨기기 |

---

## 💡 기본 사용 예시

```bash
# 기본 실행
top

# 1초마다 갱신
top -d 1

# 특정 프로세스만 감시
top -p 1234

# 특정 유저 프로세스만 표시
top -u www-data

# 유휴 프로세스 숨기고 실행
top -i

# 10번 갱신 후 종료 (스크립트용)
top -b -n 10
```

---

## 🖥️ 화면 구성 읽는 법

```
top - 15:42:01 up 10 days,  3:21,  2 users,  load average: 0.52, 0.48, 0.45
Tasks: 215 total,   1 running, 214 sleeping
%Cpu(s):  5.2 us,  1.3 sy,  0.0 ni, 92.8 id,  0.4 wa
MiB Mem :  15887.5 total,   3421.2 free,   8234.1 used,   4232.2 buff/cache
MiB Swap:   2048.0 total,   1998.4 free,     49.6 used
```

| 항목 | 설명 |
|------|------|
| `load average` | 1분 / 5분 / 15분 평균 부하 (CPU 코어 수보다 낮으면 정상) |
| `us` | 유저 영역 CPU 사용률 |
| `sy` | 커널(시스템) CPU 사용률 |
| `id` | 유휴(idle) CPU 비율 → **낮을수록 바쁜 상태** |
| `wa` | I/O 대기 비율 → **높으면 디스크 병목** |
| `buff/cache` | 캐시로 사용 중인 메모리 (필요시 해제됨) |

## ⌨️ 실행 중 단축키

| 키 | 동작 |
|----|------|
| `q` | 종료 |
| `k` | 프로세스 종료 (PID 입력) |
| `r` | 프로세스 우선순위(nice) 변경 |
| `M` | 메모리 사용량 순 정렬 |
| `P` | CPU 사용량 순 정렬 (기본) |
| `T` | 실행 시간 순 정렬 |
| `N` | PID 순 정렬 |
| `u` | 특정 유저 프로세스만 필터 |
| `d` | 갱신 주기 변경 |
| `1` | CPU 코어별 사용률 펼치기/접기 |
| `H` | 스레드 단위로 표시 |
| `c` | 명령어 전체 경로 표시 |
| `Space` | 즉시 갱신 |

## 🛠️ 실전 활용 예시

```bash
# CPU 점유 프로세스 상위 5개만 출력 (1회)
top -b -n 1 | head -17

# CPU 사용률 높은 프로세스 찾기
top -b -n 1 | sort -k9 -rn | head -10

# 결과를 파일로 저장
top -b -n 5 > top_log.txt

# 특정 프로세스 CPU/메모리 감시
top -p $(pgrep nginx | tr '\n' ',')

# watch 와 조합해서 스냅샷 반복 저장
watch -n 5 "top -b -n 1 | head -20"

# 메모리 사용 상위 프로세스 추출
top -b -n 1 | awk 'NR>7 {print $0}' | sort -k10 -rn | head -5
```

## 🔁 top vs htop 비교

| 항목 | top | htop |
|------|-----|------|
| 기본 설치 | ✅ 기본 내장 | ❌ 별도 설치 필요 |
| UI | 텍스트 기반 | 컬러 + 바 그래프 |
| 마우스 지원 | ❌ | ✅ |
| 프로세스 종료 | 키 입력 방식 | 클릭 가능 |
| 스크롤 | ❌ | ✅ |
| 추천 상황 | 서버 기본 환경 | 로컬 개발 환경 |

```bash
# htop 설치
sudo apt install htop   # Ubuntu/Debian
sudo yum install htop   # CentOS/RHEL
```

## ⚡ 자주 쓰는 조합 모음

```bash
top -d 1           # 1초 갱신 (빠른 모니터링)
top -b -n 1        # 1회 스냅샷 출력 (스크립트 연동)
top -u 유저명      # 특정 유저 프로세스만 감시
top -p PID         # 특정 프로세스만 집중 감시
top + M            # 실행 후 메모리 순 정렬 (메모리 누수 탐지)
top + 1            # CPU 코어별 사용률 확인 (멀티코어 확인)
```
