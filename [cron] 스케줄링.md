# 🕐 cron (Cron Job) 명령어 치트시트

> 특정 시간 또는 주기적으로 명령어를 자동 실행하는 리눅스/유닉스 스케줄러

---

## 📌 기본 문법

```
* * * * * 실행할_명령어
│ │ │ │ │
│ │ │ │ └── 요일 (0~7, 0과 7은 일요일)
│ │ │ └──── 월 (1~12)
│ │ └────── 일 (1~31)
│ └──────── 시 (0~23)
└────────── 분 (0~59)
```

---

## ⚙️ crontab 주요 명령어

| 명령어 | 설명 |
|--------|------|
| `crontab -e` | 현재 사용자의 크론탭 편집 |
| `crontab -l` | 현재 사용자의 크론탭 목록 출력 |
| `crontab -r` | 현재 사용자의 크론탭 삭제 |
| `crontab -u [유저명] -e` | 특정 사용자의 크론탭 편집 (root 권한) |
| `crontab -u [유저명] -l` | 특정 사용자의 크론탭 목록 출력 |

---

## ⏱️ 특수 문자

| 문자 | 설명 | 예시 |
|------|------|------|
| `*` | 모든 값 (매번) | `* * * * *` → 매 분 |
| `,` | 여러 값 지정 | `0 9,18 * * *` → 9시, 18시 |
| `-` | 범위 지정 | `0 9-18 * * *` → 9시~18시 매 시 정각 |
| `/` | 간격 지정 | `*/10 * * * *` → 10분마다 |

---

## 📅 자주 쓰는 스케줄 예시

| 표현 | 설명 |
|------|------|
| `* * * * *` | 매 1분마다 |
| `*/5 * * * *` | 5분마다 |
| `*/10 * * * *` | 10분마다 |
| `0 * * * *` | 매 시간 정각 |
| `0 0 * * *` | 매일 자정 (00:00) |
| `0 9 * * *` | 매일 오전 9시 |
| `0 9 * * 1` | 매주 월요일 오전 9시 |
| `0 9 * * 1-5` | 평일(월~금) 오전 9시 |
| `0 0 1 * *` | 매월 1일 자정 |
| `0 0 1 1 *` | 매년 1월 1일 자정 |
| `0 0 * * 0` | 매주 일요일 자정 |

---

## 🔤 특수 문자열 (@ 단축키)

| 단축키 | 동일한 표현 | 설명 |
|--------|------------|------|
| `@reboot` | - | 시스템 재부팅 시 1회 실행 |
| `@yearly` | `0 0 1 1 *` | 매년 1회 |
| `@annually` | `0 0 1 1 *` | 매년 1회 (@yearly 와 동일) |
| `@monthly` | `0 0 1 * *` | 매월 1회 |
| `@weekly` | `0 0 * * 0` | 매주 1회 (일요일) |
| `@daily` | `0 0 * * *` | 매일 1회 (자정) |
| `@midnight` | `0 0 * * *` | 매일 자정 (@daily 와 동일) |
| `@hourly` | `0 * * * *` | 매 시간 1회 |

---

## 💡 실전 예시

```bash
# 매일 새벽 2시에 백업 스크립트 실행
0 2 * * * /home/user/backup.sh

# 5분마다 서버 상태 체크
*/5 * * * * /usr/local/bin/check_server.sh

# 평일 오전 9시에 리포트 생성
0 9 * * 1-5 python3 /home/user/report.py

# 매월 1일 자정에 로그 파일 정리
0 0 1 * * rm -rf /var/log/myapp/*.log

# 재부팅 시 자동으로 서비스 시작
@reboot /home/user/start_service.sh

# 크론 실행 결과를 로그 파일에 저장
0 1 * * * /home/user/job.sh >> /var/log/job.log 2>&1

# 크론 실행 결과 출력 무시
0 1 * * * /home/user/job.sh > /dev/null 2>&1
```

---

## 📁 크론 관련 시스템 디렉토리

| 경로 | 설명 |
|------|------|
| `/etc/crontab` | 시스템 전체 크론탭 |
| `/etc/cron.d/` | 패키지별 크론 설정 파일 디렉토리 |
| `/etc/cron.hourly/` | 매시간 실행할 스크립트 |
| `/etc/cron.daily/` | 매일 실행할 스크립트 |
| `/etc/cron.weekly/` | 매주 실행할 스크립트 |
| `/etc/cron.monthly/` | 매월 실행할 스크립트 |
| `/var/spool/cron/` | 사용자별 크론탭 저장 위치 |

---

## ⚠️ 주의사항

- 크론은 기본적으로 **최소한의 환경 변수**만 로드하므로, 스크립트 내에서 절대 경로 사용을 권장
- 환경 변수가 필요하면 크론탭 상단에 명시적으로 선언

```bash
# 크론탭 상단에 환경 변수 설정
PATH=/usr/local/sbin:/usr/local/bin:/sbin:/bin:/usr/sbin:/usr/bin
SHELL=/bin/bash
MAILTO=admin@example.com   # 실행 결과를 이메일로 전송 (빈 문자열이면 전송 안 함)

0 2 * * * /home/user/backup.sh
```
