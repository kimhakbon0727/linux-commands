# 🔍 ps (Process Status) 명령어 치트시트
> 현재 실행 중인 프로세스 목록을 출력하는 리눅스/유닉스 명령어

---

## 📌 기본 문법
```
ps [옵션]
```

---

## ⚙️ 주요 옵션
| 옵션 | 설명 |
|------|------|
| `a` | 터미널에 연결된 모든 사용자의 프로세스 출력 |
| `u` | 사용자 친화적 형식으로 출력 (CPU, 메모리 등 포함) |
| `x` | 터미널에 연결되지 않은 프로세스도 출력 |
| `e` | 모든 프로세스 출력 (`-A` 와 동일) |
| `f` | 풀 포맷으로 출력 (부모-자식 관계 포함) |
| `l` | 긴 포맷으로 출력 |
| `p <PID>` | 특정 PID 프로세스 출력 |
| `u <user>` | 특정 사용자의 프로세스 출력 |
| `--forest` | 프로세스 트리 형태로 출력 |
| `-o` | 출력 컬럼 직접 지정 |

---

## 📊 출력 컬럼 설명
| 컬럼 | 설명 |
|------|------|
| `PID` | 프로세스 ID |
| `PPID` | 부모 프로세스 ID |
| `USER` | 프로세스 소유자 |
| `TTY` | 연결된 터미널 (`?` 는 터미널 없음) |
| `STAT` | 프로세스 상태 |
| `%CPU` | CPU 사용률 |
| `%MEM` | 메모리 사용률 |
| `VSZ` | 가상 메모리 크기 (KB) |
| `RSS` | 실제 사용 중인 메모리 크기 (KB) |
| `START` | 프로세스 시작 시간 |
| `TIME` | 누적 CPU 사용 시간 |
| `COMMAND` | 실행 명령어 |

---

## 🔄 STAT 프로세스 상태값
| 상태 | 설명 |
|------|------|
| `R` | 실행 중 (Running) |
| `S` | 슬립 상태 - 인터럽트 가능 (Sleeping) |
| `D` | 슬립 상태 - 인터럽트 불가 (Disk I/O 대기) |
| `T` | 정지 상태 (Stopped) |
| `Z` | 좀비 프로세스 (Zombie) |
| `<` | 높은 우선순위 |
| `N` | 낮은 우선순위 |
| `s` | 세션 리더 |
| `l` | 멀티스레드 |
| `+` | 포그라운드 프로세스 |

---

## 💡 실전 예시
```bash
# 현재 터미널의 프로세스만 출력
ps

# 가장 많이 쓰는 조합 - 모든 프로세스 상세 출력
ps aux

# 모든 프로세스 풀 포맷 출력 (부모-자식 관계 포함)
ps -ef

# 프로세스 트리 형태로 출력
ps --forest

# 혹은
ps -ef --forest

# 특정 PID 프로세스 확인
ps -p 1234

# 특정 사용자의 프로세스 확인
ps -u username

# 특정 프로세스 이름으로 검색 (grep 활용)
ps aux | grep nginx

# CPU 사용률 높은 순으로 정렬
ps aux --sort=-%cpu

# 메모리 사용률 높은 순으로 정렬
ps aux --sort=-%mem

# 상위 10개 프로세스만 출력
ps aux --sort=-%cpu | head -11

# 출력 컬럼 직접 지정
ps -eo pid,ppid,user,%cpu,%mem,command

# 특정 프로세스의 PID만 추출
ps aux | grep nginx | awk '{print $2}'
```

## 🔀 ps vs top 비교
| 항목 | ps | top |
|------|-----|-----|
| 출력 방식 | 실행 시점의 스냅샷 | 실시간 갱신 |
| 상호작용 | ❌ | ✅ |
| 스크립트 활용 | ✅ 용이 | ❌ 어려움 |
| 특정 프로세스 검색 | `grep` 조합 | `/` 검색 |
| 주 용도 | 프로세스 조회·필터링 | 시스템 전체 모니터링 |

---

## ⚠️ 주의사항
- `ps aux` 와 `ps -ef` 는 비슷하지만 출력 형식이 다름 (`aux` 는 BSD 스타일, `-ef` 는 UNIX 스타일)
- `grep` 으로 프로세스 검색 시 grep 자체 프로세스도 함께 출력될 수 있음
- 좀비 프로세스(`Z`)는 이미 종료됐지만 부모 프로세스가 회수하지 않은 상태로, `kill` 로 제거 불가
