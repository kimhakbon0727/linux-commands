# 📂 파일 목록 조회 - ls 명령어

> 디렉토리의 **파일과 폴더 목록을 출력**하는 명령어  
> 리눅스에서 가장 많이 쓰이는 기본 명령어 중 하나

---

## 📌 기본 문법

```bash
ls [옵션] [경로]
```

---

## ⚙️ 주요 옵션

| 옵션 | 설명 |
|------|------|
| `-l` | 상세 정보 표시 (권한, 소유자, 크기, 날짜) |
| `-a` | 숨김 파일(`.`으로 시작) 포함 전체 표시 |
| `-h` | 파일 크기를 사람이 읽기 쉬운 단위로 표시 (KB, MB, GB) |
| `-r` | 역순 정렬 |
| `-t` | 수정 시간 순 정렬 (최신순) |
| `-S` | 파일 크기 순 정렬 (큰 것부터) |
| `-R` | 하위 디렉토리까지 재귀적으로 표시 |
| `-i` | inode 번호 표시 |
| `-n` | 소유자/그룹을 이름 대신 UID/GID 숫자로 표시 |
| `--color` | 파일 종류별 색상 표시 (기본 활성화) |

---

## 💡 기본 사용 예시

```bash
# 현재 디렉토리 목록
ls

# 특정 경로 목록
ls /var/log

# 상세 정보 표시
ls -l

# 숨김 파일 포함
ls -a

# 상세 + 숨김 + 사람이 읽기 쉬운 크기
ls -lah

# 최신 수정 파일 순으로 정렬
ls -lt

# 파일 크기 큰 순으로 정렬
ls -lS

# 하위 디렉토리까지 전체 표시
ls -R
```

## 🖥️ 출력 화면 읽는 법

```
drwxr-xr-x  2 user group 4096 Jun 10 14:32 documents
-rw-r--r--  1 user group 1234 Jun 10 13:21 README.md
lrwxrwxrwx  1 user group   12 Jun 10 12:00 link -> /etc/hosts
```

### 파일 타입 (첫 번째 문자)

| 문자 | 의미 |
|------|------|
| `-` | 일반 파일 |
| `d` | 디렉토리 |
| `l` | 심볼릭 링크 |
| `c` | 문자 장치 파일 |
| `b` | 블록 장치 파일 |
| `p` | 파이프 |
| `s` | 소켓 |

### 권한 읽는 법

```
drwxr-xr-x
│└──────────────────── 권한 (소유자/그룹/기타)
│
└─────────── 파일 타입
```

| 구분 | 문자 | 의미 |
|------|------|------|
| 읽기 | `r` | 파일 읽기 / 디렉토리 목록 조회 가능 |
| 쓰기 | `w` | 파일 수정 / 디렉토리 파일 생성·삭제 가능 |
| 실행 | `x` | 파일 실행 / 디렉토리 진입 가능 |
| 없음 | `-` | 해당 권한 없음 |

## 🛠️ 실전 활용 예시

```bash
# 가장 많이 쓰는 조합 (상세 + 숨김 + 단위)
ls -lah

# 최근 수정된 파일 역순 (가장 최신 파일이 맨 위)
ls -lt

# 가장 오래된 파일 먼저 보기
ls -ltr

# 용량 큰 파일 먼저 정렬
ls -lSh

# 특정 확장자만 표시
ls *.log
ls *.sh

# 디렉토리만 표시
ls -d */

# 파일 개수 세기
ls | wc -l

# 숨김 파일 포함 전체 개수
ls -a | wc -l

# 결과를 파일로 저장
ls -lah > file_list.txt

# 특정 이름 패턴 검색
ls -l /var/log/*.log
```

## 🔁 ls vs 다른 명령어 비교

| 명령어 | 용도 | 특징 |
|--------|------|------|
| `ls` | 파일 목록 조회 | 빠르고 간단 |
| `ll` | `ls -l` 단축 별칭 | 일부 배포판에서만 동작 |
| `find` | 파일 검색 | 조건 검색 가능 (이름, 크기, 날짜 등) |
| `tree` | 트리 구조로 표시 | 별도 설치 필요 (`apt install tree`) |
| `du` | 디스크 사용량 확인 | 디렉토리 크기 집계 |
| `stat` | 파일 상세 메타데이터 | inode, 접근시간 등 포함 |

```bash
# tree 설치 및 사용
sudo apt install tree
tree -L 2          # 2단계 깊이까지 표시
tree -ah           # 숨김 파일 + 크기 포함
```

## ⚡ 자주 쓰는 조합 모음

```bash
ls -lah            # 상세 + 숨김 + 읽기 쉬운 크기 (가장 많이 씀)
ls -lt             # 최신 수정 파일 순 정렬
ls -ltr            # 오래된 파일 순 정렬 (로그 분석 시 유용)
ls -lSh            # 큰 파일 순 정렬 (용량 점검 시 유용)
ls -d */           # 디렉토리만 표시
ls -R              # 하위 폴더까지 전체 표시
ls *.확장자        # 특정 확장자 파일만 표시
```

## 🎨 색상 의미 (--color 활성화 시)

| 색상 | 의미 |
|------|------|
| 🔵 파란색 | 디렉토리 |
| ⚪ 흰색 | 일반 파일 |
| 🟢 초록색 | 실행 가능한 파일 |
| 🔴 빨간색 | 압축 파일 (`.tar`, `.zip` 등) |
| 🩵 하늘색 | 심볼릭 링크 |
| 🟡 노란색 | 장치 파일 |

---

> 💬 **Tip**: `ls -lah`를 `ll`로 단축 등록하면 편리해요!
> ```bash
> # ~/.bashrc 또는 ~/.zshrc에 추가
> alias ll='ls -lah'
> alias lt='ls -lt'
> alias lS='ls -lSh'
>
> # 적용
> source ~/.bashrc
> ```
